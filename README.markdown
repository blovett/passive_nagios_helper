# Passive Nagios Plugin Helper

For those people that have many passive nagios checks to run, this is for you.

## Example config:

<pre>
:services:
  :mysql:
    :description: MYSQL
    :command: check_mysql
  :mysql_slave:
    :description: MYSQL_SLAVE
    :command: check_mysql
    :args: -S
:config:
  :nagios: nagios.example.com
</pre>
